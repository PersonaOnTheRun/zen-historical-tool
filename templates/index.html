Enter your ZenCash Address in the below field. After submitting, please allow a few moments for the page to gather the data. If you encounter issues, please try using Chrome.


The goal of this tool is to provide quick access to the historical price of ZenCash at the time it was received. You will be prompted to download a .csv file containing the Date, ZenCash Amount, ZenCash Price (USD), and Value in USD for all of the address's incoming transactions. All outbound transactions are ingored. The price of ZenCash is pulled from CryptoCompare's API and the transaction data is streamed from explorer.zencash.io
